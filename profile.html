Deployment profiles¶

It is possible to automate iLO configurations with deployment profiles. With the following methods you can manage and apply such profiles.

class hpilo.Ilo
profile_desc_download(desc_name, name, description, blob_namespace=None, blob_name=None, url=None)¶

Make the iLO download a blob and create a deployment profile

profile_list()¶

List all profile descriptors

profile_delete(desc_name)¶

Delet the specified deployment profile

profile_apply(desc_name, action)¶

Apply a deployment profile

profile_apply_get_results()¶

Retrieve the results of the last profile_apply
